# 🐍 Python for Absolute Beginners – Volunteer Teaching Program  

Welcome to our **1-month Python Bootcamp** for absolute beginners, proudly conducted under the **non-profit organization icode guru**.  
This repository contains **lecture materials, code files, assignments, mini-projects, and extra resources** to help you kickstart your journey in Python programming.  

---

## 📅 Course Schedule (4 Weeks – 20 Classes, 1 Hour Each)  

### **Week 1: Python Foundations + Basic Data Structures**
**Day 1: Introduction & Setup**
- What is programming?  
- Why Python? Features & real-world uses  
- Installing Python + IDE (VS Code / Jupyter)  
- Compilation and Interpretation  
- Keywords & identifiers  
- Comments (single-line & multi-line)  
- Writing first program (Hello World)  
- `print()` function + parameters  

**Day 2: Variables & Data Types**
- Variables & naming rules  
- Methods of creating variables  
- Static vs dynamic typing  
- Data types overview  
- `type()` function  
- Taking user input with `input()`  
- Type conversion (implicit vs explicit casting)  
- Importance of indentation  

**Day 3: Operators**
- Literals  
- Arithmetic operators (+, -, *, /, //, %, **)  
- Assignment operators (=, +=, -=, etc.)  
- Comparison operators (==, !=, >, <, >=, <=)  
- Logical operators (and, or, not)  
- Operator precedence  

**Day 4: Strings**
- Creating and accessing strings  
- Adding/editing/deleting strings  
- Indexing & slicing  
- String operations (+, *, in, not in)  
- String methods & functions  
- String formatting (f-strings, `.format()`)  

**Day 5: Lists**
- Lists vs Arrays  
- Creating lists  
- Accessing/editing/deleting items  
- Indexing, slicing, stepping  
- Operations on lists (+, *, in)  
- List methods  
- Built-in list functions  

---

### **Week 2: Control Flow & Functions**
**Day 1: Conditional Statements**
- Decision control: if, elif, else  
- Nested if statements  
- Shorthand if expressions  

**Day 2: Loops (Part 1)**
- `for` loop with `range()`  
- Iterating over lists & strings  
- Nested loops  
- Loop control (break, continue, pass)  

**Day 3: Loops (Part 2) – Patterns**
- `while` loop basics  
- Combining loop + conditions  
- Printing number patterns  
- Printing star patterns  

**Day 4: Functions (Basics)**
- Built-in functions vs user-defined functions  
- Defining functions with `def`  
- Parameters vs arguments  
- Default arguments, keyword arguments  
- Variable-length arguments (*args, **kwargs)  
- Function calling & execution  
- Return statement vs no return  

**Day 5: Functions (Advanced)**
- Variable scope (local vs global)  
- Nested functions  
- Lambda functions  
- Recursion basics  

---

### **Week 3: Advanced Data Structures**
**Day 1: Tuples & Sets**
- Tuples: immutable, indexing & slicing  
- Tuple methods (count, index)  
- Sets: creating & accessing  
- Adding/removing items  
- Set operations  
- Set methods & functions  
- Frozenset  

**Day 2: Dictionaries**
- Creating dictionaries  
- Accessing values (keys, values, items)  
- Adding, updating, deleting key-value pairs  
- Dictionary operations  
- Dictionary methods & functions  
- Nested dictionaries  

**Day 3: Comprehensions**
- List comprehensions  
- Dictionary comprehensions  
- Set comprehensions  
- Conditional comprehensions  

**Day 4: Practice Day**
- Mixed problems with strings, lists, tuples, sets, dictionaries  

**Day 5: Mini Project (Data Structures)**
- Example projects:  
  - Word counter  
  - Student gradebook  
  - Shopping cart system  

---

### **Week 4: Object-Oriented Programming & Beyond**
**Day 1: OOP Basics**
- What is an object?  
- Classes & objects syntax  
- `__init__` constructor  
- Methods vs functions  
- `self` keyword  
- Multiple objects  

**Day 2: OOP Advanced**
- Magic methods (`__str__`, `__add__`, etc.)  
- Attribute creation (inside vs outside class)  
- Reference variables (pass by reference vs value)  
- Object mutability  
- Encapsulation  

**Day 3: OOP Relationships**
- Static variables vs instance variables  
- Inheritance (single, multiple, multilevel, hierarchical)  
- Method overriding & overloading  
- `super()` keyword  
- Polymorphism  
- Abstraction  

**Day 4: Modules & File Handling**
- Importing modules (`import`, `from ... import`)  
- Common modules: `math`, `random`, `datetime`  
- File handling: `open()`, `read()`, `write()`, `close()`  
- `with open()` context manager  

**Day 5: Error Handling & Final Project**
- Errors: ValueError, ZeroDivisionError, etc.  
- try-except-finally  
- Raising exceptions  
- Final project ideas:  
  - Contact Book (dict + file handling)  
  - Quiz Game  
  - To-Do List Manager
