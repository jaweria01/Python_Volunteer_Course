- Indexing, slicing, stepping  
- Operations on lists (+, *, in)  
- List methods  
- Built-in list functions

### **Week 2: Control Flow & Functions**
**Day 1: Conditional Statements**
- Decision control: if, elif, else  
- Nested if statements  
- Shorthand if expressions  

**Day 2: Loops (Part 1)**
- `for` loop with `range()`  
- Iterating over lists & strings  
- Nested loops

- Built-in functions vs user-defined functions

- Function calling & execution

**Day 5: Functions (Advanced)**
- Variable scope (local vs global)  
- Nested functions  
- Lambda functions  
- Recursion basics

### **Week 3: Advanced Data Structures**
**Day 1: Tuples & Sets**
- Tuples: immutable, indexing & slicing  
- Tuple methods (count, index)  
- Sets: creating & accessing  
- Adding/removing items  
- Set operations  
- Set methods & functions  
- Frozenset  

**Day 2: Dictionaries**
- Creating dictionaries  
- Accessing values (keys, values, items)  
- Adding, updating, deleting key-value pairs  
- Dictionary operations  
- Dictionary methods & functions  
- Nested dictionaries  

**Day 3: Comprehensions**
- List comprehensions  
- Dictionary comprehensions  
- Set comprehensions  
- Conditional comprehensions  

**Day 4: Practice Day**
- Mixed problems with strings, lists, tuples, sets, dictionaries  

**Day 5: Mini Project (Data Structures)**

  - Word counter  
  - Student gradebook  
  - Shopping cart system

### **Week 4: Object-Oriented Programming & Beyond**
**Day 1: OOP Basics**
- What is an object?  
- Classes & objects syntax  
- `__init__` constructor  
- Methods vs functions  
- `self` keyword  
- Multiple objects  

**Day 2: OOP Advanced**
- Magic methods (`__str__`, `__add__`, etc.)  
- Attribute creation (inside vs outside class)  
- Reference variables (pass by reference vs value)  
- Object mutability  
- Encapsulation  

**Day 3: OOP Relationships**
- Static variables vs instance variables  
- Inheritance (single, multiple, multilevel, hierarchical)  
- Method overriding & overloading  
- `super()` keyword  
- Polymorphism  
- Abstraction  

**Day 4: Modules & File Handling**
- Importing modules (`import`, `from ... import`)  
- Common modules: `math`, `random`, `datetime`  
- File handling: `open()`, `read()`, `write()`, `close()`  
- `with open()` context manager  

**Day 5: Error Handling & Final Project**
